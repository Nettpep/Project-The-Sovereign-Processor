# Character: ลีอาน่า เอวาลิน (Liana Evalyn) - นางเอกตัวจริง

## 📖 ข้อมูลพื้นฐาน
- **ชื่อ:** ลีอาน่า เอวาลิน (Liana Evalyn)
- **อายุ:** 22 ปี
- **ฉายา:** ธิดาแห่งความเมตตาที่โลกหลงลืม
- **ฐานะ:** บุตรีของบารอนเอวาลิน ขุนนางชั้นผู้น้อยที่ดูแลหมู่บ้านเกษตรกรรมเล็กๆ ชายขอบอาณาจักร
- **ตระกูล:** ตระกูลเอวาลิน - ไม่มีอำนาจทางการเมือง แต่ได้รับความรักจากชาวบ้านอย่างท่วมท้น
- **สถานะ:** ราชินีเคียงข้างจักรพรรดิเงา (Future Queen)
- **ที่อยู่ปัจจุบัน:** หมู่บ้านเกษตรกรรมชายขอบอาณาจักร
- **ที่อยู่ในอนาคต:** [[เมืองกราเวีย (Gravia)]]

## 👤 ลักษณะภายนอก
- **รูปร่าง:** งดงามราวกับเทพธิดาจากสรวงสวรรค์ที่หลุดออกมาจากภาพวาด
- **เส้นผม:** สีเงินบริสุทธิ์ดั่งแสงจันทร์
- **นัยน์ตา:** สีเขียวมรกตที่เปี่ยมไปด้วยความอบอุ่น
- **รอยยิ้ม:** สามารถละลายน้ำแข็งในใจของผู้คนได้เพียงแค่สบตา
- **ความงดงาม:** งามไร้ที่ติจนแม้แต่[[เนตรคัดสรร (Eye of Appraisal)]]ยังสั่นไหว

## 🎭 บุคลิกภาพ
- **อ่อนโยนแต่ไม่อ่อนแอ:** เธอเป็นคนจิตใจดี แต่ไม่ยอมแพ้ต่อความอยุติธรรม
- **จิตใจดี:** ชอบช่วยเหลือผู้ยากไร้และเหล่าพลทหารที่บาดเจ็บโดยไม่เกี่ยงชนชั้น
- **ความเชื่อ:** เธอเชื่อใน "คุณค่าของความเป็นคน" มากกว่า "ยศถาบรรดาศักดิ์"
- **ความเมตตา:** มีความเมตตาที่บริสุทธิ์ผุดผ่อง ไม่มีเจตนาร้ายซ่อนเร้น

## 💔 จุดเริ่มต้นความผูกพัน: "หยาดน้ำในทะเลทราย"

### บริบท
ในช่วงที่[[ไคราส (Kairos)]]ยังเป็นพลทหารชั้นต่ำและถูกกลั่นแกล้งจนบาดเจ็บปางตาย (ก่อนจะถูกเนรเทศ) ลีอาน่า คือคนเดียวที่เข้ามาทำแผลให้เขาในโรงเรือนทหารที่ซ่อมซ่อ โดยไม่รังเกียจเนื้อตัวที่เปื้อนโคลนและเลือด

### คำพูดสำคัญ
> "ท่านเจ็บมากไหม? อย่าเพิ่งยอมแพ้นะคะ... โลกนี้อาจจะใจร้ายกับท่าน แต่ขอให้ท่านรู้ว่ายังมีคนที่เห็นความดีในตัวท่านอยู่" — ลีอาน่า

### ผ้าเช็ดหน้า
- **ความสำคัญ:** ผ้าเช็ดหน้าผืนเล็กที่ลีอาน่ามอบให้ไคราส
- **สถานะ:** เป็น "สมบัติชิ้นเดียว" ที่ไคราสพกติดตัวไปที่[[เมืองกราเวีย (Gravia)]]
- **ความหมาย:** เป็นเหตุผลเดียวที่ทำให้เขายังเหลือความเป็นมนุษย์อยู่ในใจ

## 🌧️ ฉากการพบกันครั้งแรก: "หยาดฝนและแสงเทียน" (The Goddess in the Rain)

### บรรยากาศ
หลังจากไคราสถูก[[วาเลเรียส_ดาร์คฟอร์ด|วาเลเรียส]] (พี่ชายของ[[เซราฟีน่า_ดาร์คฟอร์ด|เซราฟีน่า]]) สั่งให้ทหารรุมซ้อมจนปางตายด้วยข้อหา "บังอาจตีตนเสมอท่านหญิง" ร่างของเขาถูกโยนทิ้งไว้ในกองโคลนข้างกำแพงเมือง ไคราสนอนมองฟ้าที่มืดมิดด้วยสายตาที่ว่างเปล่า ความโกรธแค้นและความเจ็บปวดทำให้เขากำลังจะสูญเสียสติไป

### การปรากฏตัว
ในขณะที่สติของเขาเริ่มเลือนลาง ท่ามกลางเสียงฝน เขากลับได้ยินเสียงฝีเท้าที่แผ่วเบาและกลิ่นหอมอ่อนๆ เหมือนดอกไม้ป่า แสงสว่างจากตะเกียงดวงเล็กๆ ค่อยๆ เคลื่อนเข้ามาใกล้ พร้อมกับเงาของใครบางคนที่มาบดบังสายฝนที่ร่วงหล่นใส่หน้าเขา

เขาลืมตาขึ้นอย่างยากลำบาก และนั่นเป็นครั้งแรกที่เขาได้เห็นเธอ... ลีอาน่า

### บทสนทนาและท่าทาง
เธอย่อตัวลงในกองโคลนโดยไม่ห่วงว่าชุดสีขาวสะอาดของเธอจะเปื้อน มือที่เรียวบางและอบอุ่นประคองใบหน้าที่อาบไปด้วยเลือดของไคราสขึ้นมาอย่างเบามือ

**ลีอาน่า:** (เสียงสั่นเครือด้วยความสงสาร) "ท่านทหาร... อดทนไว้นะคะ อย่าเพิ่งหลับตานะ..."

ไคราสพยายามจะดันตัวออกตามสัญชาตญาณทหารที่ระแวงไปทั่ว แต่สัมผัสที่อ่อนโยนนั้นทำให้เขาหยุดชะงัก ลีอาน่าใช้ผ้าเช็ดหน้าสะอาดซับเลือดที่หางตาของเขาอย่างถะนุถนอม สายตาของเธอที่มองมาไม่มีความรังเกียจทหารชั้นต่ำอย่างเขาสักนิด มีเพียงความเมตตาที่บริสุทธิ์

**ไคราส:** (เสียงแหบพร่า) "ทำไม... เจ้าเป็นขุนนาง... ทำไมต้องมาช่วยขยะอย่างข้า..."

**ลีอาน่า:** (ยิ้มเศร้าๆ แต่ดงามดั่งเทพธิดา) "ไม่มีใครเป็นขยะในสายตาของพระเจ้าหรอกค่ะ ท่านเป็นมนุษย์ที่กำลังเจ็บปวด และข้าก็เป็นเพียงมนุษย์คนหนึ่งที่อยากช่วยท่านเท่านั้นเอง"

### การใช้ "เนตรคัดสรร" ครั้งแรก: การตื่นของพลัง
ในวินาทีนั้น ไคราสที่หัวใจกำลังจะแตกสลายได้เผลอใช้พลังพิเศษมองไปที่เธอ และสิ่งที่เขาเห็นทำให้เขาตกตะลึงจนลืมความเจ็บปวด:

#### วินาทีที่พลังตื่น
ในจังหวะที่นิ้วมืออันอ่อนโยนของลีอาน่าสัมผัสที่ใบหน้าของไคราส โลกที่เคยเป็นสีเทากลับกลายเป็นสีดำสนิทชั่วครู่ ก่อนจะมีตัวอักษรสีทองเรืองแสงวิ่งผ่านนัยน์ตาของเขาเหมือน Code คอมพิวเตอร์!

```
[ระบบผู้ชักใย: เริ่มการซิงโครไนซ์วิญญาณ]
[เงื่อนไขครบถ้วน: สภาวะจิตใจแตกสลาย 99% / สัมผัสจากผู้มีจิตบริสุทธิ์ 100%]
[ปลดล็อกความสามารถ: เนตรคัดสรร (Eye of Appraisal)]
```

#### ข้อมูลที่เห็น
ทันใดนั้น ภาพตรงหน้าเปลี่ยนไป ไคราสไม่ได้เห็นแค่ผู้หญิงคนหนึ่ง แต่เห็น:
- **สายใยแห่งโชคชะตา:** สายใยแห่งโชคชะตาและค่าสเตตัสที่ซ่อนอยู่ของลีอาน่า
- **จุดตาย:** เห็น "จุดตาย" ของศัตรู
- **ความชั่วร้าย:** เห็น "ความชั่วร้าย" ในใจของพวกขุนนางที่เดินอยู่ไกลๆ

```
[ข้อมูลเป้าหมาย: ลีอาน่า เอวาลิน]

สถานะ: บุตรีบารอนชั้นผู้น้อย (ชนชั้นสูงธรรมดา)

ความงดงาม: SSS (ระดับเทพธิดาสวรรค์)

ความจิตใจดี: MAX (บริสุทธิ์ไร้ที่ติ)

ความรู้สึกต่อท่าน: สงสารและปรารถนาดี (100%)
```

#### ความหมายของการตื่น
- **ลีอาน่า คือ "กุญแจ":** ลีอาน่า คือ "กุญแจ" สำคัญที่ทำให้ระบบของพระเอกทำงานได้
- **ความดีงาม:** ความดีงามของเธอคือสิ่งที่พระเอกต้องการปกป้อง
- **การเปลี่ยนแปลง:** จากนี้ไป ไคราสจะใช้ "ดวงตาคู่นี้" ชักใยโลกทั้งใบให้กราบกรานเขา

ลีอาน่าหยิบขวดยาเล็กๆ ออกมาทาที่แผลให้เขา พร้อมกับสวมผ้าคลุมกันฝนของเธอให้เขาแทน

**ลีอาน่า:** "ข้าต้องรีบกลับแล้ว พี่ชายข้ากำลังตามหา... รับสิ่งนี้ไว้นะคะ (เธอยัดถุงเงินเล็กๆ และผ้าเช็ดหน้าใส่มือเขา) ขอให้ท่านมีชีวิตอยู่ต่อนะคะท่านทหาร โลกนี้อาจจะมืดมน แต่ข้าจะสวดอ้อนวอนให้ท่านได้เจอแสงสว่างในสักวัน"

เธอลุกขึ้นและหายไปในม่านฝน ทิ้งให้ไคราสนอนกำผ้าเช็ดหน้าผืนนั้นไว้แน่น

## 👑 บทบาทในฐานะ "ราชินีเคียงข้างจักรพรรดิเงา"

### การคุ้มครอง
เมื่อไคราสเริ่มสร้างมหาอำนาจที่[[เมืองกราเวีย (Gravia)]] เขาจะส่ง[[เอเลน_ผู้บัญชาการหน่วยรบพิเศษ|เอเลน]] (แม่ทัพเงา) ไปคอยคุ้มครองเธอลับๆ จากเงามืด

### การเชิญมา
เมื่อถึงเวลาที่เหมาะสม เขาจะเชิญเธอมาอยู่ที่เมืองของเขา เพื่อให้เธอเป็น "หัวใจ" ของประชาชน ในขณะที่เขาเป็น "สมอง" และ "กรงเล็บ"

### บทบาท
- **หัวใจ:** ลีอาน่าเป็น "หัวใจ" ของประชาชน
- **สมองและกรงเล็บ:** ไคราสเป็น "สมอง" และ "กรงเล็บ"
- **ความสมดุล:** ทั้งสองคนสร้างความสมดุลให้กับอาณาจักร
