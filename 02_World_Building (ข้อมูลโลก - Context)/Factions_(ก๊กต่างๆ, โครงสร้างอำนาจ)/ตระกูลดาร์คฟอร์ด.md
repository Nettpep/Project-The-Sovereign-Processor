# ตระกูลดาร์คฟอร์ด (Darkford)

## 📖 ข้อมูลพื้นฐาน
- **ชื่อ:** ตระกูลดาร์คฟอร์ด (Darkford)
- **สัญลักษณ์ประจำตระกูล:** สิงโตดำคาบหอกหัก
- **อำนาจในมือ:** การทหารและป้องกัน
- **บทบาททางการเมือง:** คุมกองทัพหลักและกระทรวงกลาโหม
- **สถานะ:** หนึ่งใน 5 ตระกูลมหาอำนาจแห่ง[[อาณาจักรอาเธลการ์ด|อาณาจักรอาเธลการ์ด]]
- **ประมุข:** [[มัลฟาส_ดาร์คฟอร์ด|ดยุก มัลฟาส ดาร์คฟอร์ด]] - มหาเสนาบดีฝ่ายกลาโหม

## 🎭 ลักษณะของตระกูล
- **ขึ้นชื่อเรื่อง:** ความเด็ดขาด เลือดเย็น และยึดมั่นในเกียรติยศชนชั้นเหนือสิ่งอื่นใด
- **บุคลิก:** น่าเกรงขาม เย่อหยิ่ง และยึดมั่นในระบบชนชั้น
- **ความสำคัญ:** ตระกูลที่วางแผนเนรเทศ[[ไคราส (Kairos)]]ไป[[เมืองกราเวีย (Gravia)]]

## 👥 สมาชิกครอบครัว

### ประมุขตระกูล
- **[[มัลฟาส_ดาร์คฟอร์ด|ดยุก มัลฟาส ดาร์คฟอร์ด]]** - ประมุขตระกูล และมหาเสนาบดีฝ่ายกลาโหม
  - บทบาท: วางแผนเนรเทศไคราส
  - Status: INT: A, LDR: S+, POL: S
  - ความลับ: กำลังวางแผนชักใยให้กษัตริย์สละราชสมบัติ

### แม่ทัพใหญ่
- **[[มหาอุปราชมัลพาส|มหาอุปราชมัลพาส]]** - เงามืดแห่งบัลลังก์ (พี่ชายของมัลฟาส)
  - บทบาท: มหาอุปราชและแม่ทัพใหญ่หลักของอาณาจักร
  - Status: LDR: S, POL: A+, INT: A+, STR: A
  - คู่ต่อกร: [[เซนอส_แม่ทัพใหญ่สายทำลายล้าง|เซนอส]] - แม่ทัพใหญ่สายทำลายล้างของไคราส

### บุตรชาย
- **[[วาเลเรียส_ดาร์คฟอร์ด|วาเลเรียส ดาร์คฟอร์ด]]** - บุตรชายคนโต ผู้สืบทอดตระกูล
  - บทบาท: เย่อหยิ่ง ทะนงในฝีมือดาบ
  - Status: STR: A+, AGI: A, LDR: B
  - จุดอ่อน: ใจร้อนและประเมินศัตรูต่ำเกินไป
  - เป้าหมาย: เป้าหมายหลักที่ไคราสจะใช้ทำลายตระกูลจากภายใน

- **[[จูเลียน_ดาร์คฟอร์ด|จูเลียน ดาร์คฟอร์ด]]** - บุตรชายคนเล็ก
  - บทบาท: อ่อนโยนและรักศิลปะ
  - Status: INT: S, CHA: A+, Luck: A
  - ความหมาย: อาจเป็น "หมาก" ที่ไคราสเลือกชักใยให้ขึ้นมาเป็นผู้นำตระกูลคนใหม่

### บุตรี
- **[[เซราฟีน่า_ดาร์คฟอร์ด|เซราฟีน่า ดาร์คฟอร์ด]]** - บุตรีลำดับที่หนึ่ง
  - บทบาท: รักแรกของไคราสที่หักหลัง
  - Status: CHA: S, INT: B+, ความจริงใจ: 10/100
  - ปัจจุบัน: หมั้นหมายกับเจ้าชายลำดับที่ 2

## ⚔️ ความขัดแย้งกับไคราส

### การเนรเทศ
- มัลฟาสเป็นคนวางแผนเนรเทศไคราสไปเมืองกราเวีย
- มองว่าไคราสคือ "สิ่งเจือปน" ที่จะมาทำให้สายเลือดชนชั้นสูงแปดเปื้อน

### ฉากดราม่าในวันที่ถูกเนรเทศ
- **มัลฟาส:** ยืนมองจากระเบียงสูง
- **วาเลเรียส:** หัวเราะเยาะอยู่ข้างสนาม และเหยียบตราทหารของไคราส
- **เซราฟีน่า:** ยืนจิบน้ำชาโดยไม่แม้แต่จะชายตามามองไคราสที่ถูกใส่กุญแจมือ

### คำพูดของวาเลเรียส
> "จงไปตายในดินแดนโสโครกนั่นเถอะไคราส... อย่าให้ชื่อของเจ้ากลับมาทำให้ตระกูลดาร์คฟอร์ดต้องมัวหมองอีก"

## 🎯 กลยุทธ์ของไคราส

### เป้าหมาย
- **ทำลายตระกูลจากภายใน:** ใช้วาเลเรียสเป็นเป้าหมายหลัก
- **ควบคุมตระกูล:** ใช้จูเลียนเป็น "หมาก" เพื่อควบคุมตระกูล
- **แก้แค้น:** ทำให้ตระกูลเห็นว่าไคราสไม่ได้เป็นเพียง "พลทหารชั้นต่ำ"

### กลยุทธ์
- ใช้จุดอ่อนของวาเลเรียส (ใจร้อนและประเมินศัตรูต่ำเกินไป)
- ชักใยให้จูเลียนขึ้นมาเป็นผู้นำตระกูลคนใหม่
- ล้มล้างมัลฟาสและวาเลเรียส

## 📝 หมายเหตุสำคัญ

ตระกูลดาร์คฟอร์ดคือตระกูลที่วางแผนเนรเทศไคราส และจะเป็นเป้าหมายหลักในการแก้แค้นของไคราส
