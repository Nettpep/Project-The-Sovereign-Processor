# Bloodline: เนตรคัดสรร (Eye of Appraisal)

## 🔮 ข้อมูลพื้นฐาน
- **ชื่อ:** เนตรคัดสรร (Eye of Appraisal)
- **เจ้าของ:** [[ไคราส (Kairos)]]
- **ความหายาก:** Unique - หนึ่งเดียวในโลก
- **ประเภท:** ความสามารถในการรับรู้และประเมินค่า

## ⚡ การตื่นของพลัง (Power Awakening)

### เงื่อนไขการปลดล็อก
พลังนี้ถูกปลดล็อกเมื่อ:
- **สภาวะจิตใจแตกสลาย:** 99% (ไคราสถูกซ้อมจนปางตายและถูกโยนทิ้ง)
- **สัมผัสจากผู้มีจิตบริสุทธิ์:** 100% (การสัมผัสจาก[[ลีอาน่า_เอวาลิน|ลีอาน่า เอวาลิน]])
- **การซิงโครไนซ์วิญญาณ:** วิญญาณจากโลกอนาคตซิงโครไนซ์กับร่างของไคราส

### ฉากการตื่น
- **สถานที่:** กองโคลนข้างกำแพงเมือง ท่ามกลางพายุฝน
- **สถานะ:** ไคราสนอนจมกองเลือด ใจเริ่มดับวูบ
- **การปรากฏตัว:** [[ลีอาน่า_เอวาลิน|ลีอาน่า]]ปรากฏตัวพร้อมแสงตะเกียง
- **การสัมผัส:** ในจังหวะที่นิ้วมืออันอ่อนโยนของลีอาน่าสัมผัสที่ใบหน้าของเขา
- **การเปลี่ยนแปลง:** โลกที่เคยเป็นสีเทากลับกลายเป็นสีดำสนิทชั่วครู่
- **การปลดล็อก:** ตัวอักษรสีทองเรืองแสงวิ่งผ่านนัยน์ตาของเขาเหมือน Code คอมพิวเตอร์

### ระบบการทำงาน
```
[ระบบผู้ชักใย: เริ่มการซิงโครไนซ์วิญญาณ]
[เงื่อนไขครบถ้วน: สภาวะจิตใจแตกสลาย 99% / สัมผัสจากผู้มีจิตบริสุทธิ์ 100%]
[ปลดล็อกความสามารถ: เนตรคัดสรร (Eye of Appraisal)]
```

### ความหมายของการตื่น
- **ลีอาน่า คือ "กุญแจ":** [[ลีอาน่า_เอวาลิน|ลีอาน่า]] คือ "กุญแจ" สำคัญที่ทำให้ระบบของพระเอกทำงานได้
- **ความดีงาม:** ความดีงามของเธอคือสิ่งที่พระเอกต้องการปกป้อง
- **การเปลี่ยนแปลง:** จากนี้ไป ไคราสจะใช้ "ดวงตาคู่นี้" ชักใยโลกทั้งใบให้กราบกรานเขา

### สิ่งที่เห็นครั้งแรก
เมื่อพลังตื่น ไคราสเห็น:
- **สายใยแห่งโชคชะตา:** สายใยแห่งโชคชะตาและค่าสเตตัสที่ซ่อนอยู่ของลีอาน่า
- **จุดตาย:** เห็น "จุดตาย" ของศัตรู
- **ความชั่วร้าย:** เห็น "ความชั่วร้าย" ในใจของพวกขุนนางที่เดินอยู่ไกลๆ

## 💫 ความสามารถหลัก

### 1. การมองเห็นค่าสถานะ (Status Value Perception)
- **ความสามารถ:** มองเห็น "ค่าสถานะ" ของบุคคลได้
- **ข้อมูลที่เห็น:**
  - ระดับความสามารถปัจจุบัน
  - ศักยภาพที่ยังไม่ได้พัฒนา
  - สถานะทางสังคมที่แท้จริง
  - ระดับอิทธิพลและอำนาจ

### 2. การมองเห็นความสามารถแฝง (Hidden Talent Detection)
- **ความสามารถ:** ตรวจจับความสามารถที่ซ่อนเร้นของบุคคล
- **การใช้งาน:**
  - ค้นหา "เพชรในตม" (คนที่มีความสามารถแต่ยังไม่ถูกค้นพบ)
  - ระบุผู้ที่มีพรสวรรค์ด้านการวิจัยเทคโนโลยี AI/Automations
  - ค้นหาผู้เชี่ยวชาญในสาขาต่างๆ ที่ยังไม่ถูกใช้งาน
  - ประเมินศักยภาพในการพัฒนา

### 3. การตรวจจับความทรยศ (Betrayal Detection)
- **ความสามารถ:** ตรวจจับคนทรยศหรือผู้ที่มีเจตนาไม่ดี
- **การใช้งาน:**
  - เห็นเจตนาที่แท้จริงของบุคคล
  - ตรวจจับผู้ที่แฝงตัวมาเป็นสายลับ
  - ระบุผู้ที่มีความเสี่ยงในการทรยศ
  - มองทะลุการแสดงออกที่หลอกลวง

### 4. การประเมินศักยภาพ (Potential Assessment)
- **ความสามารถ:** ประเมินศักยภาพและความสามารถที่ซ่อนเร้น
- **การใช้งาน:**
  - ประเมินว่าบุคคลสามารถพัฒนาได้มากแค่ไหน
  - ระบุจุดแข็งและจุดอ่อนที่แท้จริง
  - คาดการณ์ผลลัพธ์ในการฝึกฝนหรือพัฒนา
  - ประเมินความเหมาะสมกับตำแหน่งหรืองาน

## 🎯 การใช้งานในเรื่อง

### การรวบรวมขุนพล
- **[[เว่ยหลัน_อัครเสนาธิการ|เว่ยหลัน]]:** ค้นพบว่ามีความสามารถด้านกลยุทธ์ระดับตำนาน
- **[[เบิร์น_ผู้อำนวยการฝ่ายอาวุธสงคราม|เบิร์น]]:** ค้นพบพรสวรรค์ด้านการประดิษฐ์และกลไก
- **[[เซียนั่ว_เจ้ากรมมหาดไทย|เซียนั่ว]]:** ค้นพบความสามารถในการต่อรองระดับโลก
- **[[กาเรธ_แม่ทัพใหญ่|กาเรธ]]:** ค้นพบความแข็งแกร่งและความภักดี
- **[[เอเลน_ผู้บัญชาการหน่วยรบพิเศษ|เอเลน]]:** ค้นพบความสามารถในการลอบเร้นและการสังหาร

### การป้องกันการแทรกซึม
- ตรวจจับสายลับที่ขุนนางส่งมา
- มองทะลุเจตนาของผู้ที่พยายามเข้ามาในเมือง
- ระบุผู้ที่มีความเสี่ยงในการทรยศ

### การจัดการความสัมพันธ์
- มองทะลุเจตนาของรักแรกที่พยายามกลับมา
- ระบุผู้ที่ซื่อสัตย์และไว้วางใจได้
- ประเมินความเหมาะสมของพันธมิตร

## ⚠️ ข้อจำกัด

### ข้อจำกัดทางกายภาพ
- ต้องมองเห็นบุคคลด้วยตาเปล่า (หรือผ่านกระจก/น้ำ)
- ไม่สามารถใช้ได้กับวัตถุหรือสถานที่
- ต้องใช้สมาธิในการใช้งาน

### ข้อจำกัดทางจิตใจ
- การใช้งานต่อเนื่องอาจทำให้เหนื่อยล้า
- ไม่สามารถเห็นอนาคตได้ เพียงแต่ประเมินศักยภาพ
- อาจจะถูกหลอกได้ถ้าผู้ถูกมองมีพลังพิเศษที่ซ่อนเร้น

### ข้อจำกัดทางสังคม
- ไม่สามารถบอกผู้อื่นได้ว่าเห็นอะไร (อาจจะถูกมองว่าเป็นคนบ้า)
- ต้องใช้ข้อมูลที่เห็นอย่างระมัดระวัง
- อาจจะทำให้ไว้วางใจคนยากเกินไป

## 🔍 รายละเอียดการทำงาน

### ข้อมูลที่แสดง
เมื่อใช้เนตรคัดสรร ไคราสจะเห็น:
- **ค่าสถานะ:** ตัวเลขหรือระดับที่แสดงความสามารถ
- **สี/แสง:** สีหรือแสงที่แสดงประเภทของความสามารถ
- **สัญลักษณ์:** สัญลักษณ์ที่แสดงความสามารถพิเศษ
- **เส้นทาง:** เส้นทางที่แสดงศักยภาพในการพัฒนา

### ตัวอย่างการใช้งาน
- **เพชรในตม:** เห็นแสงสีทองที่สว่างจ้าแต่ถูกบดบัง
- **คนทรยศ:** เห็นสีแดงหรือสีดำที่แสดงเจตนาไม่ดี
- **ผู้เชี่ยวชาญ:** เห็นสัญลักษณ์ที่แสดงความเชี่ยวชาญในสาขา
- **ศักยภาพสูง:** เห็นเส้นทางที่ยาวและสว่าง

## 🎭 ผลกระทบต่อตัวละคร

### ผลกระทบเชิงบวก
- สามารถรวบรวมคนที่มีความสามารถได้อย่างมีประสิทธิภาพ
- ป้องกันการทรยศและการแทรกซึมได้
- สร้างทีมที่แข็งแกร่งและซื่อสัตย์

### ผลกระทบเชิงลบ
- อาจจะไว้วางใจคนยากเกินไป
- อาจจะมองคนเป็นเพียง "ทรัพยากร" แทนที่จะเป็นมนุษย์
- อาจจะรู้สึกโดดเดี่ยวเพราะเห็นความจริงที่คนอื่นไม่เห็น
